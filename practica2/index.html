ESP32 Wokwi Sensores

- Resumen -

En esta práctica se llevará a cabo la identificación y prueba del funcionamiento de sensores digitales, analógicos e inteligentes en el ESP32. Se configurará el entorno de desarrollo en Wokwi y Arduino IDE y se implementarán ejemplos prácticos para comprender el funcionamiento de distintos sensores: un botón como sensor digital, un potenciómetro como sensor analógico y un sensor DHT22 como sensor inteligente.

Para concluir esta práctica, se programará la detección de movimiento a través de un sensor PIR.

- Introducción -

Un sensor es un dispositivo que detecta el cambio en el entorno y responde a alguna salida en el otro sistema. Un sensor convierte un fenómeno físico en un voltaje analógico medible (o, a veces, una señal digital) convertido en una pantalla legible para humanos o transmitida para lectura o procesamiento adicional.

Para ello, se utilizarán herramientas como Vokwi o Arduino IDE para programar el ESP32 y realizar pruebas con diferentes tipos de sensores. Se trabajará con un botón para detectar señales digitales, un potenciómetro para leer valores analógicos y un sensor DHT22 para medir temperatura y humedad.

- Desarrollo -

En esta sección, se detallará el proceso completo para la simulación y construcción del circuito utilizando sensores digitales, analógicos e inteligentes con el ESP32. Se explicará cada etapa, desde la configuración del entorno de desarrollo en Wokwi o Arduino IDE, hasta la implementación del código y la conexión de los sensores.

Sensor Digital

Los sensores digitales operan de forma binaria, proporcionando únicamente dos estados posibles, como activado o desactivado. Su lectura se realiza a través de un GPIO configurado como entrada digital, permitiendo detectar cambios en su estado.

Botón

En este código se configura el pin 23 como salida para controlar un LED, y el pin 34 como entrada para leer el estado de un botón. En la función loop(), que se ejecuta en un bucle infinito, se lee el estado del botón y, si está presionado, el LED se enciende. Si el botón no está presionado, el LED se apaga. Esto permite controlar el encendido del LED de manera manual a través del botón. 

                                int led=23;
                                int button=34;
                                int buttonState=0;
                                void setup() {
                                    // put your setup code here, to run once:
                                    pinMode(23,OUTPUT);
                                    pinMode(34,INPUT);
                                }

                                void loop() {
                                    buttonState=digitalRead(button);
                                    if(buttonState==1){
                                        digitalWrite(led,HIGH);
                                    }
                                    if (buttonState==0){
                                        digitalWrite(led,LOW);
                                    }
                                }

Sensor Analógico

Los sensores analógicos proporcionan valores de salida continuos que varían en un rango de voltaje, dependiendo de la resolución del ADC del microcontrolador. En el caso del ESP32 tiene un ADC de 12 bits, los valores van de 0 a 4096. Para realizar la lectura de estos valores, se utiliza un GPIO con un canal analógico, permitiendo medir la variación de voltaje del sensor y obtener datos más precisos.

Potenciómetro

En este código se configura el pin 23 como salida para controlar un LED, y el pin 34 como entrada para leer el valor de un potenciómetro. En la función loop(), que se ejecuta en un bucle infinito, se lee el valor analógico del potenciómetro. Si el valor leído es mayor o igual a 1024, el LED se enciende, de lo contrario, el LED se apaga. 

                                int led=23;
                                int pot=34;
                                
                                int potValue=0;
                                
                                void setup() {
                                  // put your setup code here, to run once:
                                  pinMode(23,OUTPUT);
                                  pinMode(34,INPUT);
                                }
                                
                                void loop() {
                                  
                                  potValue=analogRead(pot);
                                
                                  if(potValue>=1024){
                                    digitalWrite(led,HIGH);
                                  }
                                  else{
                                    digitalWrite(led,LOW);
                                  }
                                
                                }

Sensor Inteligente

Los sensores inteligentes son dispositivos que van más allá de las simples lecturas digitales o analógicas, ya que su funcionamiento o complejidad requiere el uso de protocolos de comunicación, puertos específicos del microcontrolador o medición de tiempos. Estos sensores suelen ser más sofisticados.

DHT22

En este código se configura el pin 23 como salida para controlar un LED y se utiliza el sensor DHT22 conectado al pin 33 para medir la temperatura y la humedad. En la función loop(), que se ejecuta continuamente, el sensor lee la temperatura y la humedad cada segundo. Estos valores se envían al monitor seria. Si la humedad medida es mayor o igual al 50%, el LED se enciende, y si la humedad es menor a ese valor, el LED se apaga. 

                                int led=23;
                                int temperatura=0;
                                int humedad=0;
                                #include 
                                DHT dht(33,DHT22);

                                void setup() {
                                    // put your setup code here, to run once:
                                    pinMode(23,OUTPUT);
                                    dht.begin();
                                    Serial.begin(115200);
                                }

                                void loop() {
                                delay(1000);
                                temperatura=dht.readTemperature();
                                humedad=dht.readHumidity();

                                Serial.print("Temperatura:");
                                Serial.println(temperatura);
                                Serial.print("Humedad:");
                                Serial.println(humedad);

                                if(humedad>=50){
                                    digitalWrite(led,HIGH);
                                }
                                else{
                                    digitalWrite(led,LOW);
                                }
                            }

- Conclusiones -

A lo largo de esta práctica, se exploró el uso de entradas y salidas digitales del ESP32, permitiendo implementar diferentes funciones básicas de control. Se programaron circuitos que facilitaron desde la activación de un LED mediante distintas condiciones lógicas hasta la generación de parpadeos.

Mediante el uso de estructuras condicionales, se pueden diseñar sistemas básicos de control, como encender un LED con un solo botón (lógica OR) o exigir la activación simultánea de dos botones (lógica AND). Estos conceptos son fundamentales para desarrollar aplicaciones más complejas en sistemas embebidos e IoT.
