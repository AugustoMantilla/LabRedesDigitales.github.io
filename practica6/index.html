Configuración de servidor para Red Local

Augusto Mantilla Molina
Ingeniería Mecatrónica
Jorge Eduardo Limón Balderas
Ingeniería Mecatrónica
Fecha: Abril 16, 2025
XAMPP Cables de red

- Resumen -

En la presente práctica se creó un servidor local utilizando XAMPP y se probó el acceso al servidor con distintos dispositivos diferentes a la computadora que opera el programa.

- Introducción -

En esta práctica, se implementó un servidor local utilizando la herramienta XAMPP, que integra un servidor Apache, permitiendo alojar sitios web de manera sencilla en una computadora personal. El objetivo principal fue habilitar el acceso a dicho servidor desde distintos dispositivos conectados a la misma red, como teléfonos móviles, tabletas o computadoras adicionales, para verificar la correcta configuración y accesibilidad del servicio. Esto implicó la identificación de la dirección IP del servidor, la configuración de puertos en el firewall y la comprobación de conectividad.

- Materiales -

  • Computadora como servidor
  • Tres dispositivos diferentes

- Desarrollo -

La práctica comenzó con la descarga del paquete XAMPP desde el sitio oficial de Apache Friends. Una vez descargado, se ejecutó el instalador como administrador para asegurar los permisos necesarios en el sistema. Durante la instalación, se seleccionaron los módulos predeterminados, destacando Apache, que es el servidor web utilizado para gestionar las solicitudes HTTP. Desde la configuración avanzada del firewall, se creó una nueva regla de entrada para permitir el tráfico TCP a través del puerto 80.

Para permitir que otros dispositivos accedieran al servidor, se identificó la dirección IP local de la computadora que ejecuta XAMPP. Esto se logró abriendo la terminal de comandos y ejecutando el comando ipconfig. En los resultados, se localizó la sección correspondiente a la interfaz de red activa y se anotó la dirección IPv4.

Con la dirección IP del servidor y el puerto habilitado, se procedió a realizar la prueba de conectividad desde tres dispositivos diferentes conectados a la misma red Wi-Fi. En cada uno de ellos, se abrió un navegador web y se ingresó la dirección IP obtenida anteriormente.

- Conclusiones -

La práctica permitió comprender y aplicar los pasos necesarios para instalar y configurar un servidor local utilizando XAMPP en un entorno Windows. Se logró habilitar correctamente el servicio Apache, identificar la dirección IP del servidor para verificar el acceso desde distintos dispositivos conectados a la misma red.
