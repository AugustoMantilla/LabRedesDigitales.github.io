ESP32 Wokwi Sensores

- Resumen -

En esta práctica, se implementa la plataforma ThingSpeak para la recopilación y monitoreo de datos en la nube, utilizando un módulo ESP32-C6. Se configura la conectividad WiFi para la transmisión de datos en tiempo real y se integran sensores digitales, analógicos e inteligentes. Además, se incorpora un sensor PIR para la detección de movimiento, permitiendo su visualización en un tablero público en ThingSpeak. Se emplea Arduino IDE para la programación y pruebas iniciales, y se realizan simulaciones en Wokwi antes del montaje en hardware, garantizando su correcto funcionamiento. Esta práctica refuerza conocimientos en IoT, redes inalámbricas y procesamiento de datos en la nube.

- Introducción -

La interconectividad entre sensores y plataformas IoT es clave para la automatización y supervisión remota de sistemas electrónicos. ThingSpeak es una plataforma en la nube que permite visualizar y analizar datos en tiempo real, facilitando su integración con microcontroladores como el ESP32-C6.

En esta práctica, se configura la conexión del ESP32-C6 a redes WiFi, permitiendo la transmisión de datos de sensores digitales, analógicos e inteligentes a un tablero público en ThingSpeak para su monitoreo a distancia. Además, se realizan pruebas en Wokwi antes de la implementación física para asegurar un correcto funcionamiento.

Escaneo WiFi

En la función setup(), se inicia la comunicación serial a 115200 baudios y se configura el ESP32 en modo estación (WIFI_STA). En la función loop(), el ESP32 escanea las redes WiFi cercanas usando WiFi.scanNetworks(), y muestra en el monitor serial la cantidad de redes encontradas junto con su SSID (nombre), la intensidad de la señal (RSSI) y si la red está protegida o es abierta. Luego, espera 5 segundos antes de realizar un nuevo escaneo. 

                                #include "WiFi.h"

                                void setup() {
                                Serial.begin(115200);
                                Serial.println("Inicializando WiFi...");
                                
                                WiFi.mode(WIFI_STA);
                                Serial.println("Configuración terminada!");
                                }

                                void loop() {
                                Serial.println("Escaneando...");
                                
                                int n = WiFi.scanNetworks();
                                Serial.println("Escaneo terminado!");
                                
                                if (n == 0) {
                                    Serial.println("No se encontraron redes.");
                                } else {
                                    Serial.println();
                                    Serial.print(n);
                                    Serial.println(" redes encontradas");
                                    
                                    for (int i = 0; i < n; ++i) {
                                    Serial.print(i + 1);
                                    Serial.print(": ");
                                    Serial.print(WiFi.SSID(i));
                                    Serial.print(" (");
                                    Serial.print(WiFi.RSSI(i));
                                    Serial.print(")");
                                    Serial.println((WiFi.encryptionType(i) == WIFI_AUTH_OPEN) ? " " : "*");
                                    
                                    delay(10);
                                    }
                                }
                                Serial.println("");
                                delay(5000);
                                }

Sensor Digital

En la función setup(), se configura la conexión WiFi al comenzar con los parámetros de red, y se establece la comunicación serial. Luego, en la función loop(), se lee el estado del botón y, en función de su valor, se enciende o apaga el LED. Además, si el ESP32 está conectado a WiFi, se construye y envía una solicitud HTTP GET a ThingSpeak para actualizar el valor del botón. 

                                    #include 
                                    #include 
                                    
                                    int boton = 4;
                                    int led = 1;
                                    
                                    int status_boton = 0;
                                    
                                    String url = "https://api.thingspeak.com/update?";
                                    String api_key = "NZUJ6QMBRY8VJSTS";
                                    String sensor = "field1";
                                    
                                    void setup() {
                                      pinMode(led, OUTPUT);
                                      pinMode(boton, INPUT);
                                    
                                      delay(1000);
                                      Serial.begin(115200);
                                        
                                      WiFi.begin("Guty", "123456789");
                                    
                                      while ((WiFi.status() != WL_CONNECTED)) {
                                        delay(500);
                                        Serial.print(".");
                                      }   
                                      Serial.println("WiFi conectado");
                                    }
                                    
                                    void loop() {
                                      status_boton = digitalRead(boton);
                                    
                                      if (status_boton == 1) {
                                        digitalWrite(led, HIGH);
                                      } else {
                                        digitalWrite(led, LOW);
                                      }
                                    
                                      if ((WiFi.status() == WL_CONNECTED)) {
                                        HTTPClient http;
                                    
                                        String message = url + "api_key=" + api_key + "&" + sensor + "=" + String(status_boton);
                                        http.begin(message);
                                    
                                        int httpCode = http.GET();
                                        Serial.println(message);
                                        Serial.print("Código GET: ");
                                        Serial.println(httpCode);
                                    
                                        http.end();
                                      }
                                    
                                      delay(3000);
                                    }

Sensor Analógico

En setup(), se configura la conexión WiFi, la comunicación serial y el pin del LED como salida. En loop(), se lee el valor del potenciómetro y, si es mayor o igual a 1024, el LED se enciende; de lo contrario, se apaga. Si hay conexión WiFi, se construye y envía una solicitud HTTP GET para actualizar el valor en field2 de ThingSpeak, mostrando en el monitor serial el código de respuesta para verificar el éxito de la actualización.

                                    }

Sensor Inteligente

Este código utiliza un sensor DHT22 para medir la temperatura y la humedad, y envía los datos a ThingSpeak a través de una conexión WiFi en un ESP32. En la función setup(), se inicializa el sensor, se configura un LED en el pin 4 como salida y se establece la conexión WiFi. En loop(), se leen los valores de temperatura y humedad. Si la humedad supera el 50%, el LED se enciende; de lo contrario, se apaga. Si el ESP32 está conectado a WiFi, los datos de humedad se envían a ThingSpeak. 

                                    #include 
                                    #include 
                                    #include 
                                    
                                    int led = 4;
                                    DHT dht(5, DHT22);
                                    
                                    int valor_temperatura = 0;
                                    int valor_humedad = 0;
                                    
                                    String url = "https://api.thingspeak.com/update?";
                                    String api_key = "NZUJ6QMBRY8VJSTS";
                                    String sensor = "field5";
                                    
                                    void setup() {
                                      pinMode(led, OUTPUT);
                                      dht.begin();
                                      
                                      delay(1000);
                                      Serial.begin(115200);
                                        
                                      WiFi.begin("Guty", "123456789");
                                    
                                      while ((WiFi.status() != WL_CONNECTED)) {
                                        delay(500);
                                        Serial.print(".");
                                      }   
                                      Serial.println("WiFi conectado");
                                    }
                                    
                                    void loop() {
                                      delay(1000);
                                    
                                      valor_temperatura = dht.readTemperature();
                                      valor_humedad = dht.readHumidity();
                                    
                                      Serial.print("Temperatura: ");
                                      Serial.println(valor_temperatura);
                                      Serial.print("Humedad: ");
                                      Serial.println(valor_humedad);
                                    
                                      if (valor_humedad >= 50) { 
                                        digitalWrite(led, HIGH);
                                      } else {
                                        digitalWrite(led, LOW);
                                      }
                                    
                                      if ((WiFi.status() == WL_CONNECTED)) {
                                        HTTPClient http;
                                        String message = url + "api_key=" + api_key + "&" + sensor + "=" + String(valor_humedad);
                                        http.begin(message);
                                        int httpCode = http.GET();
                                        Serial.println(message);
                                        Serial.print("Código GET: ");
                                        Serial.println(httpCode);
                                        http.end();
                                      }
                                    
                                      delay(1000);
                                    }

PIR

En la función setup(), se configura el pin del LED como salida y el pin del sensor PIR como entrada, se establece la conexión WiFi. En la función loop(), se lee el estado del sensor PIR, y si se detecta movimiento, el LED se enciende. Si no hay movimiento, el LED se apaga y se notifica el fin del movimiento. Si el ESP32 está conectado a WiFi, se construye y envía una solicitud HTTP GET a ThingSpeak para actualizar el valor del sensor en field4. 

                                    #include 
                                    #include 
                                    
                                    int led = 1;
                                    int PIR = 4;
                                    
                                    int pirState = LOW;
                                    int val = 0;
                                    
                                    String url = "https://api.thingspeak.com/update?";
                                    String api_key = "NZUJ6QMBRY8VJSTS";
                                    String sensor = "field4";
                                    
                                    void setup() {
                                      pinMode(led, OUTPUT);
                                      pinMode(PIR, INPUT);
                                    
                                      delay(1000);
                                      Serial.begin(115200);
                                        
                                      WiFi.begin("Guty", "123456789");
                                    
                                      while ((WiFi.status() != WL_CONNECTED)) {
                                        delay(500);
                                        Serial.print(".");
                                      }   
                                      Serial.println("WiFi conectado");
                                    }
                                    
                                    void loop() {
                                      delay(1000);
                                    
                                      val = digitalRead(PIR);
                                    
                                      if (val == HIGH) {
                                            digitalWrite(led, HIGH);
                                            if (pirState == LOW) {
                                                Serial.println("¡Movimiento detectado!");
                                                pirState = HIGH;
                                            }
                                        } else {
                                            digitalWrite(led, LOW);
                                            if (pirState == HIGH) {
                                                Serial.println("¡Movimiento finalizado!");
                                                pirState = LOW;
                                            }
                                        }
                                    
                                      if ((WiFi.status() == WL_CONNECTED)) {
                                        HTTPClient http;
                                    
                                        String message = url + "api_key=" + api_key + "&" + sensor + "=" + String(pirState);
                                        http.begin(message);
                                    
                                        int httpCode = http.GET();
                                        Serial.println(message);
                                        Serial.print("Código GET: ");
                                        Serial.println(httpCode);
                                    
                                        http.end();
                                      }
                                    
                                      delay(1000);
                                    }
