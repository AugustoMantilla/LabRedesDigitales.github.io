- Resumen -

En esta práctoca vamos a ver la configuración y uso de las entradas y salidas digitales en un módulo NodeMCU ESP32. Se inicia con la instalación y configuración del IDE de Arduino para programar el NodeMCU. Luego, se revisa el pinout y los GPIO disponibles.

Posteriormente, se desarrollan diferentes programas para controlar un LED, incluyendo encendido y apagado simple, parpadeo y control de su estado mediante botones. También se exploran condiciones lógicas OR y AND para el control del LED con dos botones.

- Introducción -

El ESP32 es un microcontrolador con conectividad WiFi y Bluetooth, utilizado en proyectos de IoT y automatización. Su arquitectura, múltiples pines GPIO y soporte para diversos protocolos de comunicación lo hacen ideal para aplicaciones avanzadas.

En esta práctica, se explorará el uso de sus entradas y salidas digitales, comenzando con la configuración del IDE de Arduino para programarlo. Se revisará el pinout del ESP32 y sus GPIO. Posteriormente, se desarrollarán programas básicos para controlar un LED, incluyendo su encendido y apagado mediante diferentes códigos, retardos y el uso de botones. Además, se implementarán condiciones lógicas OR y AND para gestionar el LED con dos botones.

Encendido

En este código se configura el pin 23 como salida, permitiendo que este pin controle el LED conectado. En la función loop(), que se ejecuta en un bucle infinito, en donde el pin en estado alto. Como no hay retardos ni cambios en el estado del pin, este permanecerá siempre en alto, manteniendo activado cualquier dispositivo conectado.

                                void setup() {
                                    // put your setup code here, to run once:
                                    pinMode(23,OUTPUT);
                                }

                                void loop() {
                                    digitalWrite(led, HIGH);
                                }

Parpadeo

En este código se configura el pin 23 como salida, y lo alterna entre alto y bajo con un retardo de 500 ms en cada cambio, lo que hace que un LED conectado a ese pin parpadee de forma continua. 

                                int led=23;
                                void setup() {
                                    // put your setup code here, to run once:
                                    pinMode(23,OUTPUT);
                                }

                                void loop() {
                                digitalWrite(led, HIGH);
                                delay(500);
                                digitalWrite(led, LOW);
                                delay(500);
                                }

Botón

En este código el pin 23 como salida para controlar un LED y el pin 34 como entrada para leer el estado de un botón. En el loop(), se verifica continuamente el estado del botón con digitalRead(34). Si el botón está presionado, el LED se enciende; si el botón no está presionado, el LED se apaga. 

                                int led=23;
                                int button=34;
                                int buttonState=0;
                                void setup() {
                                    // put your setup code here, to run once:
                                    pinMode(23,OUTPUT);
                                    pinMode(34,INPUT);
                                }

                                void loop() {
                                    buttonState=digitalRead(button);

                                    if(buttonState==1){
                                        digitalWrite(led,HIGH);
                                    }
                                    if (buttonState==0){
                                        digitalWrite(led,LOW);
                                    }
                                }
